骰点服务

支持多种TRPG规则系统的骰点，包括COC、DND、无限流、赛博服克红、FFG系战锤规则、魔导书大战、BRP、LOM诡秘及诡秘琉璃版、最终物语、北欧奇谭、喵苏鲁、忍神、永夜后日谈、撞邪、暗夜刀锋、餐云卧石、魔法猫猫救世界、绿色三角洲、黑暗世界、双重十字、共鸣性怪异、暗影狂奏等多种规则。

AI对话

与骰娘进行有趣的AI对话互动，可以回答问题、聊天或提供建议。Miss Foxsan采用活泼可爱的风格，而霜尾游则采用半古风半毒舌的风格，满足不同玩家的喜好。

娱乐功能

内置多种娱乐小游戏和互动功能，包括特色牌堆（疯狂星期四、COJ名场面、一日三餐、随机冒险等）、点歌功能、表情包制作等多种娱乐功能。

好感度系统

与骰娘建立好感度，解锁更多互动。通过日常互动和签到提升好感度，解锁特殊对话和功能，增强跑团体验。

Team功能

为管理员和主持人提供的团队管理工具，可以更方便地管理跑团成员、安排活动和发布公告，减少管理工作量。

历史记录

详细的跑团历史记录和数据统计功能，包括log日志时间和条数记录、长期骰点统计、logstat记录等，帮助玩家回顾和分析跑团数据。

单人跑团模组

提供单人跑团体验（需私聊），允许玩家在没有其他玩家的情况下体验跑团乐趣，适合新手学习和老手快速体验。

跑团辅助功能

提供各种实用的跑团辅助工具，如角色卡管理、自定义骰点表达式、快捷指令设置等，提高跑团效率和体验。

如何使用

从添加骰子到探索高级功能，分步骤指南帮助你快速上手

1

添加骰子到群聊/服务器

选择你喜欢的平台，将Miss Foxsan或霜尾游添加到你的QQ群或Discord服务器。

小提示

在QQ上，你需要先添加骰娘为好友，然后将她邀请到群聊中。在Discord上，只需点击“添加到服务器”按钮并选择你的服务器即可。

2

设置角色和基本参数

使用基本命令创建你的角色并设置属性值。

基本命令示例
.help - 查看帮助信息
.st 力量:50 敏捷:60 意志:70 - 设置属性值
.pc - 查看当前角色卡
.nn 角色名称 - 设置角色名称
小提示

不同的规则系统可能有不同的属性设置方式。使用 .help 规则名 查看特定规则的帮助。

3

使用基本骰点命令

学习基本的骰点命令，开始你的跑团之旅。

基本骰点命令
.r - 扔一个骰子 (1d100)
.r 3d6 - 扔三个六面骰
.ra 力量 - 进行属性检定
.sc 1/1d3 - 理智检定(仅COC)
.ri 敏捷 - 带有奖惩的检定(仅DND)
4

探索高级功能

当你熟悉了基本命令后，可以开始探索更多高级功能。

记录跑团日志

使用 .log on 开始记录跑团日志，.log off 停止记录。

团队管理

使用 .team create 创建团队，.team add @用户 添加成员。

自定义骰点

使用 .set 命令设置自定义骰点表达式和快捷指令。

好感度系统

使用 .favor 查看当前好感度，.sign 每日签到提升好感度。

小提示

使用 .help advanced 查看更多高级命令和功能。不同的规则系统可能有特定的高级功能。
